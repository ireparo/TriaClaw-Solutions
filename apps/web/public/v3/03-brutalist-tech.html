# 01. CENTRALITA

STATUS: ● ONLINE

Numeración nacional, IVR multilingüe y transcripción en streaming. Cada llamada queda como evento estructurado en el CRM antes de colgar.

  • IVR ES/CA con detección de intención
  • Grabación + transcripción Whisper
  • Transferencia inteligente a humano
  • Buzón con resumen + acción sugerida

# 02. AGENTS

STATUS: ● ONLINE

Agentes IA en WhatsApp, web y email. Cualifican, agendan citas y escalan a humano cuando hace falta. Sin scripts rígidos.

  • WhatsApp Business API oficial
  • Memoria por cliente y por hilo
  • Tools: agenda, catálogo, FAQ, ticket
  • Hand-off humano con un emoji
> ACCESS_MODULE

# 03. CRM

STATUS: ● ONLINE

Un único timeline por cliente: llamadas, mensajes, tickets, presupuestos y firmas. Sin duplicar datos entre apps.

  • Pipeline visual + estados custom
  • Plantillas de presupuesto + e-firma
  • Reportes diarios en automático
  • API + webhooks para integrar
